INSTRUCTIONS FOR “Are You Cognizant Ready”

 

IMPORTANT: Read this before proceeding with your “Are You Cognizant Ready” attempt! This IS NOT like “Certify yourself” or “Sharpen your skills” sessions.
 

“Are You Cognizant Ready” is an assessment of the skill and knowledge, covering all the 7 modules you learnt earlier namely, “Technology Fundamentals, Operating systems, Programming Fundamentals, Object oriented concepts, Ansi SQL, Software Engineering and Software Testing”

NOTE:

  1. If you click on the “Launch” button, it will be counted as an attempt! We will not encourage any requests for additional attempts.

  2. You cannot pause the test clock once you have attempted. So please ensure that you are available and online for 1.5 hours to take this test.

  3. Please do not open multiple tabs, as this will affect your test score.

  4. The test will automatically close after 1.5 hours, so please save and submit your answers regularly.

  5. You can enroll / cancel any assessments, 3hrs before the assessment start time.
